Find element's index in C array

int index_in_int_array(int *array, int array_length, int searched) {
  int i;

  for (i = 0; i < array_length; i++)
    if (array[i] == searched) return i;

  return -1;
}

Parameters:

  • array: the array in which you want to search an element
  • array_length: the length of the array (i.e. the number of elements)
  • searched: the element to search

Returned value:

The function returns the index of the searched element if it exists in the array, otherwise -1.

Explanation:

This function loops over an array (here is an array of integers) and return i when the element at index i in array is found, so the for loop is interrupted.
If the array is completely visited, it means that the searched element doesn't exist in array. Then, the function returns -1.
