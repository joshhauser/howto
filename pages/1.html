Generate random number with inclusive interval (JS)

function inclusiveRand(min, max) {
  min = Math.ceil(min);
  max = Math.floor(max);
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

Parameters:

  • min: the start of the interval
  • max: the end of the interval

Returned value:

The function returns a random number, between min and max.
