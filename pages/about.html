About

This website contains different tips for things that we often need as developers, but we often forget. Creating this project was a way to centralize all the small things that are simple but easily forgettable.

It's made HTML, CSS and JS.

Here are the frameworks used for this project:

  • PaperCSS for a minimalist style
  • highlight.js for syntax highlighting in tips
